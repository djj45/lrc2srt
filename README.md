### 功能介绍

将`utf-8`或者`ansi`编码的`lrc`歌词转换为`srt`歌词，前后时间连续，标出歌词持续时间大于8秒的歌词（可能为间奏），最后一句歌词持续时间默认为8秒，支持网易云音乐、酷我音乐和qq音乐的`lrc`歌词，支持批量转换，把歌词和程序放在一起运行，即可转换当前目录下的所有`lrc`歌词
